# 📚 Pesquisa de Referências - EduTrack AI

Esta pasta contém as referências de design e interface selecionadas diretamente no Figma Community. O objetivo é servir de inspiração visual e estrutural para o desenvolvimento do projeto EduTrack AI.

---

## 🎨 Referência 1: Student App (Design Mobile)
* **Link para o Figma:** https://www.figma.com/community
* **Por que escolhi esta referência?** * **Interface Limpa:** O ecrã de Login tem um design minimalista e moderno que facilita a entrada do utilizador.
  * **Acompanhamento de Notas:** A organização dos cartões escuros com grandes percentagens visuais (como 85% e 95%) é excelente para mostrar o rendimento escolar de forma rápida.
  * **Foco no Aluno:** O ecrã de "Focus Mode" com temporizador dá uma ótima ideia de funcionalidade que podemos integrar no EduTrack AI para ajudar na produtividade dos estudantes.

## 🎨 Referência 2: Educational Dashboard - Edulog (Painel Web)
* **Link para o Figma:** https://www.figma.com/community
* **Por que escolhi esta referência?**
  * **Painel em Dark Mode:** O contraste do Modo Escuro com elementos em azul e rosa vibrantes deixa a interface tecnológica e agradável para uso prolongado.
  * **Gráficos de Performance:** O gráfico de linhas para analisar o desempenho ("Performance") é ideal para o módulo de Inteligência Artificial do EduTrack AI exibir a evolução do aluno.
  * **Organização Geral:** O menu lateral esquerdo facilita a navegação entre as aulas ("Classes") e documentos, enquanto a barra lateral direita organiza as atividades futuras e o calendário de forma impecável.
