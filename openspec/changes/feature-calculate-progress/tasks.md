# Tarefas de implementação para calculate_progress

## Objetivo
Planejar a criação de um script Python que calcule a porcentagem de progresso e retorne JSON.

## Tarefas
1. Criar o arquivo `scripts/calculate_progress.py`.
2. Implementar a função `calculate_progress(completed, total)` que retorna a porcentagem de progresso.
3. Tratar o caso em que `total` é igual a zero para evitar divisão por zero.
4. Serializar o resultado como JSON antes de retorná-lo.
