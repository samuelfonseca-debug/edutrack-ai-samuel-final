# Contexto do Sistema

## Visão Geral
Este projeto é um backend de gestão académica construído com XanoScript e Desenvolvimento Guiado por Especificações.

## Objetivo
Permitir que os utilizadores registem e giram as suas disciplinas académicas e dados relacionados.

## Entidades Principais
- Users (Autenticação nativa do Xano)
- Subjects (Disciplinas académicas associadas a um utilizador)

## Relacionamentos
- Cada disciplina (Subject) pertence a um utilizador (User).
